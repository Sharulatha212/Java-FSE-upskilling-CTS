Responsive Column Layout

Resize the browser window to see how columns respond:
Mobile: 1 column | Tablet: 2 columns | Desktop: 3 columns

Column 1

This column will:

  • Take full width on mobile
  • Take half width on tablet
  • Take one-third width on desktop

Column 2

This column will:

  • Stack below on mobile
  • Move right on tablet
  • Stay in middle on desktop

Column 3

This column will:

  • Stack at bottom on mobile
  • Start new row on tablet
  • Complete row on desktop
